Kebijakan Privasi

Terakhir diperbarui: 12 Maret 2026

Aplikasi AlquranKita menghargai privasi pengguna. Dokumen Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pengguna ketika menggunakan aplikasi kami.

Dengan menggunakan aplikasi AlquranKita, Anda dianggap telah membaca dan menyetujui kebijakan privasi ini.

1. Informasi yang Kami Kumpulkan

Kami dapat mengumpulkan beberapa jenis informasi berikut:

a. Informasi Akun

Saat pengguna masuk atau menggunakan fitur tertentu, kami dapat mengumpulkan informasi seperti:

  • Nama pengguna
  • Email
  • ID pengguna unik

Informasi ini digunakan untuk mengelola akun dan memastikan fitur aplikasi dapat berjalan dengan baik.

b. Data Penggunaan

Kami dapat mengumpulkan data penggunaan aplikasi seperti:

  • Interaksi dengan fitur aplikasi
  • Partisipasi dalam event atau kuis
  • Jawaban yang dikirimkan dalam event

Data ini digunakan untuk menjalankan fitur aplikasi secara optimal.

c. Informasi Perangkat

Aplikasi dapat secara otomatis mengumpulkan informasi seperti:

  • Jenis perangkat
  • Versi sistem operasi
  • ID perangkat anonim

Informasi ini digunakan untuk meningkatkan stabilitas dan performa aplikasi.

2. Bagaimana Kami Menggunakan Informasi

Informasi yang dikumpulkan digunakan untuk:

  • Menyediakan dan menjalankan fitur aplikasi
  • Mengelola akun pengguna
  • Menyimpan jawaban event atau kuis
  • Mengirim pemberitahuan terkait aplikasi
  • Meningkatkan kualitas dan performa aplikasi

3. Penyimpanan dan Keamanan Data

Kami berusaha melindungi data pengguna dengan langkah-langkah keamanan yang wajar. Data dapat disimpan menggunakan layanan cloud pihak ketiga yang memiliki standar keamanan yang baik.

Meskipun demikian, perlu diingat bahwa tidak ada metode transmisi data melalui internet yang 100% aman.

4. Berbagi Informasi dengan Pihak Ketiga

Kami tidak menjual atau menyewakan data pribadi pengguna kepada pihak ketiga. Namun, aplikasi dapat menggunakan layanan pihak ketiga untuk mendukung operasional aplikasi, seperti layanan:

  • Autentikasi pengguna
  • Penyimpanan database
  • Notifikasi

Layanan tersebut hanya digunakan secara terbatas untuk menjalankan fungsi aplikasi.

5. Notifikasi

Aplikasi dapat mengirimkan notifikasi kepada pengguna, misalnya:

  • Pengumuman event
  • Informasi pemenang
  • Pembaruan aplikasi

Pengguna dapat menonaktifkan notifikasi melalui pengaturan perangkat kapan saja.

6. Privasi Anak

Aplikasi ini tidak secara khusus ditujukan untuk anak di bawah usia 13 tahun. Kami tidak secara sengaja mengumpulkan data pribadi anak-anak.

Jika orang tua atau wali mengetahui bahwa anak mereka memberikan informasi pribadi, silakan hubungi kami untuk penghapusan data tersebut.

7. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan diumumkan melalui aplikasi atau halaman kebijakan privasi ini.
